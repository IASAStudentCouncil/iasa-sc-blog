Курси на літо 2023

26 липня 2023 р. · 1 хв. читання
Анастасія Пижук
Редакторка статей та інтерв'юерка

Хей! Залишилося не так багато до кінця літа, тому саме час приділити увагу саморозвитку. Наша команда підготувала перелік корисних курсів, аби ІПСАшники не сиділи склавши лапки.

  1. GENESIS GROWTH WEEK (Дедлайн реєстрації: 30 липня).

  2. Спеціалізація Machine Learning.

  3. Natural Language Processing in TensorFlow.

  4. Git для розподіленої розробки програмного забезпечення.

  5. Developing Back-End Apps with Node.js and Express.

  6. Introduction to Game Theory.

  7. Building Web Applications in PHP.

  8. Mobile Development and JavaScript.

  9. Спеціалізація Android App Development.

  10. Python для DS, штучного інтелекту й розробки.

  11. Data Analytics for Lean Six Sigma.

  12. Data Visualization and Communication with Tableau.

  13. SQL для науки про дані.

  14. Основи Kubernetes.

  15. Основи Linux.
